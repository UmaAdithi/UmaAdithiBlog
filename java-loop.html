Understanding Loops in Java

In Java, loops are used to execute a block of code repeatedly. The three main types of loops in Java are:

  • For Loop
  • While Loop
  • Do-While Loop

Example of a For Loop


public class ForLoopExample {
    public static void main(String[] args) {
        for(int i = 0; i < 5; i++) {
            System.out.println(i);
        }
    }
}

In this example, the loop will print the numbers 0 to 4. The loop starts at 0 and continues as long as the value of i is less than 5.
